## 产品概述

### 产品定位

企业级AI原生协同平台是一个基于"两个底座 + 两个平台"架构的智能协同工作系统：

- **两个底座**: Agent OS（Claude Agent SDK） + 本体数据底座（Palantir风格）
- **两个平台**: 
  - **平台1**: 面向业务角色的开箱即用执行平台
  - **平台2**: 面向产品/技术角色的配置管理平台

### 核心价值主张

通过**多角色智能协同**（人-人、人-AI、AI-AI）来完成复杂业务，并在**0-1过程中沉淀经验**，实现**2-N的高效复用**，最终形成"越用越智能"的正向飞轮效应。

### 核心创新点

1. **"领域智伴"概念**: 融合领域Agent的专业性与智伴协作者的人性化
2. **动态Skill机制**: 配置时生成模板，使用时动态调整
3. **双轨经验沉淀**: 私有沉淀（即时生效）+ 平台发布（审核后共享）
4. **业务流程驱动本体**: 产品经理配置业务流程 → AI自动生成本体数据
5. **智伴分级提醒**: 直接回复、浮窗提示、角标提醒，智能而不打扰

## 核心功能概览

### 平台1（业务执行平台）

平台1是业务用户的日常工作空间，提供以下核心能力：

- **项目空间管理**: 创建、管理业务项目，邀请成员协作
- **CUI对话引擎**: 自然语言交互，@功能唤起智伴或成员
- **GUI工作台**: 内置文档编辑器、白板、思维导图等工具
- **智伴AI**: 
  - 主动性引擎（分级提醒）
  - 推荐引擎（经验、人员、工具）
- **多角色协作**: 实时同步、Thread讨论、全局通知
- **分层记忆**: 项目内记忆、跨项目记忆、用户偏好记忆
- **经验管理**: 私有沉淀、平台经验市场、搜索与安装

### 平台2（配置管理平台）

平台2是企业知识沉淀和AI能力进化的"大脑"，提供以下核心能力：

- **价值领域配置器**: 
  - 创建/管理价值领域
  - 可视化业务流程编排
  - AI辅助生成流程草稿
- **本体编辑器**: 知识图谱可视化、手动编辑对象/关系
- **Skill与工具管理器**: 
  - Skill自动生成与测试
  - MCP工具注册
- **经验审核中心**: 审核平台1用户提交的经验发布申请
- **分析与洞察仪表盘**: 数据可视化、运营监控、效果分析

### Agent OS底座

基于Claude Agent SDK构建的AI执行核心，提供以下核心能力：

- **LLM核心**: 多LLM后端支持、动态模型路由、Prompt模板管理
- **任务规划器**: 单步/多步任务规划、动态重规划、失败恢复
- **工具执行器**: 动态调用、参数校验、同步/异步执行
- **沙箱环境**: Docker隔离、代码执行、受限的文件/网络访问
- **上下文管理器**: 滑动窗口、向量检索、自动摘要、Token监控

### 本体数据底座

Palantir风格的知识图谱，提供以下核心能力：

- **核心模型**: 对象、属性、关系、逻辑、行动五大元素
- **自动生成**: 从平台2的业务流程配置自动生成本体结构
- **实例捕获**: 在平台1的业务执行中自动创建对象实例
- **知识提取**: 从经验沉淀中提取新的对象/关系类型
- **版本控制**: 本体模型的版本管理与迁移

## 实施路线图

### 第一阶段：MVP版本 (0-3个月)

**目标**: 验证核心业务闭环，跑通数据飞轮的最小化版本。

**核心场景**: 客户代表在平台1上，完成对一个客户的基本需求分析，并形成结构化报告。经验可被私有沉淀。

**交付成果**: 可用的平台1和平台2 Web应用、两个底座的核心API服务、MVP版本用户手册。

### 第二阶段：标准版本 (3-9个月)

**目标**: 完善数据飞轮，实现知识在组织内的共享与复用。

**核心场景**: 用户可以将成功经验发布到平台，经审核后形成标准Skill，其他用户可调用复用。

**交付成果**: 功能完善的平台1和平台2、完整的经验闭环、与核心外部系统的集成、运营监控仪表盘。

### 第三阶段：企业增强版 (9-18个月)

**目标**: 提升平台的深度智能和企业级服务能力，从"提效工具"向"决策大脑"演进。

**核心场景**: 通过数据分析发现业务规律，优化业务流程，实现预测性分析和自动决策。

**交付成果**: 具备决策智能的AI原生协同平台、强大的数据分析能力、完善的平台治理、丰富的开发者生态。
